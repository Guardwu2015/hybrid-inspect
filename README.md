# hybrid-inspect

hybrid-inspect是建立在anyproxy基础上的，web调试工具，采用lerna进行依赖管理，共包含两个包client和inspector。

+ client
    + electron的桌面应用，负责启动anyproxy，inspector的服务
+ inspector
    + 基于[VorlonJS](https://github.com/MicrosoftDX/Vorlonjs)项目并进行深度定制，并且在界面及交互方便进行重构
